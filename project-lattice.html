문제 상황

데이터는 참고 수준, 운영은 감에 좌우

저가 커피 매장은 빠른 회전율과 가격 경쟁력을 중심으로 운영되지만, 실제 매장 운영은 여전히 점주의 경험과 직관에 크게 의존하고 있었습니다. PoS를 통해 매출과 판매 데이터는 지속적으로 축적되고 있었지만, 고객 특성이나 시간대, 외부 환경과 같은 맥락 정보와 연결되지 않아 데이터가 실질적인 의사결정에 활용되지 못하는 한계가 있었습니다.

문제 정의

데이터가 '정책'에 머물고, '행동'으로 이어지지 않는 문제

기존 매장 데이터는 매출, 판매량, 주문 건수 등 개별 수치 중심으로 제공되고 있어, 현재 상황을 해석하거나 이후 행동으로 연결하기 어려운 구조였습니다. 어떤 고객이 주요 매출을 만드는지, 어떤 시간대에 어떤 메뉴가 잘 팔리는지, 특정 상황에서 어떤 전략을 적용해야 하는지에 대한 정보가 부재했습니다. 결국 문제의 본질은 데이터의 부재가 아니라, 데이터가 의사결정과 행동으로 연결되지 않는 구조에 있었습니다.

리서치 및 목표

저가 커피 환경에서는 '개인화'보다 '빠른 선택'이 중요

스타벅스와 같은 프리미엄 커피 브랜드는 개인화 추천을 통해 고객 경험을 강화하지만, 저가 커피 고객은 방문 전 이미 메뉴를 결정하고 빠르게 주문하는 패턴을 보입니다. 키오스크 체류 시간은 짧고, 새로운 메뉴 탐색에 대한 니즈보다 빠르고 합리적인 선택을 더 중요하게 생각하는 특징이 있습니다.

고객에게는 메뉴 선택 과정의 부담을 줄이고 빠른 주문을 가능하게 하는 구조를 제공하고, 관리자에게는 매장 운영에 필요한 핵심 인사이트를 직관적으로 제공하여 의사결정을 지원하는 것을 목표로 했습니다.

궁극적으로는 데이터 활용을 통해 주문 속도, 객단가, 운영 효율을 동시에 개선하고, 감에 의존하던 운영 방식을 데이터 기반 의사결정 구조로 전환하는 것을 핵심 목표로 설정했습니다.

문제 해결 1

고객데이터와 결제데이터를 결합하여 ‘매장 운영 인사이트’ 제공

이러한 문제를 해결하기 위해, 데이터를 단순히 수집하거나 보여주는 것이 아니라 사용자의 선택과 관리자의 의사결정을 직접적으로 지원하는 구조로 설계했습니다. 고객 측면에서는 메뉴를 추천하는 방식이 아니라 데이터 기반으로 노출 순서를 재구성하여 자연스럽게 선택을 유도하는 구조를 설계했습니다.

관리자 측면에서는 매출과 고객 데이터를 통합하여 단순 수치가 아닌 해석 가능한 정보로 제공하고, 특정 상황에서 어떤 행동을 해야 하는지까지 연결되는 인사이트 중심의 대시보드를 설계했습니다. 이를 통해 데이터가 '확인용 정보'가 아닌 '행동을 유도하는 도구'로 기능하도록 구조를 전환했습니다.

문제 해결 2

매장 맞춤형 인사이트 제공

매장마다 모든 원칙을 동일하게 따를 수 없습니다. 어떤 지역에 위치한 매장인지에 따라 고객 구성과 판매 패턴이 달라지기 때문입니다.

관리자 대시보드에서는 매출, 고객 특성, 시간대별 패턴, 상품별 성과를 통합하여 시각화하고, 단순 데이터 확인이 아니라 문제 발생 지점과 개선 방향을 바로 파악할 수 있도록 설계했습니다. 특히 변화율, 비교 구조, 핵심 지표 요약 등을 통해 데이터 해석 과정을 최소화하고, 즉각적인 판단과 실행이 가능한 구조를 구현했습니다.

문제 해결 3

키오스크와 대시보드를 연결한 데이터 기반 운영 시스템 구축

본 솔루션은 키오스크와 관리자 대시보드를 하나의 데이터 흐름으로 연결하는 구조로 설계했습니다. 키오스크에서는 고객의 선택 데이터와 외부 환경 데이터를 기반으로 메뉴 노출 구조를 동적으로 변경하고, 이를 통해 빠른 선택을 유도할 수 있도록 했습니다.

관리자 대시보드에서는 매출, 고객 특성, 시간대별 패턴, 상품별 성과를 통합하여 시각화하고, 단순 데이터 확인이 아니라 문제 발생 지점과 개선 방향을 바로 파악할 수 있도록 설계했습니다. 특히 변화율, 비교 구조, 핵심 지표 요약 등을 통해 데이터 해석 과정을 최소화하고, 즉각적인 판단과 실행이 가능한 구조를 구현했습니다.

결과

감 중심 운영에서 데이터 기반 의사결정 구조의 전환

프로젝트를 통해 고객은 메뉴를 더 빠르게 선택할 수 있는 경험을 제공받았고, 매장은 주문 속도와 회전율을 개선할 수 있는 기반을 마련했습니다. 또한 메뉴 노출 구조 개선을 통해 자연스러운 업셀링이 가능해지면서 객단가 상승 효과도 기대할 수 있는 구조를 설계했습니다.

관리자 관점에서는 매출과 고객 데이터를 기반으로 주요 고객군과 문제 구간을 명확히 파악할 수 있게 되었으며, 이를 통해 메뉴 구성, 프로모션, 운영 전략을 보다 정밀하게 수립할 수 있는 환경이 구축되었습니다. 결과적으로 본 프로젝트는 단순한 UI 개선을 넘어, 데이터를 중심으로 매장 운영 방식 자체를 전환하는 기반을 설계한 프로젝트로 정의할 수 있습니다.
